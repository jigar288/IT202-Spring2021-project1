Home Page

This app display data about land inventory that is owned by the city of chicago in a nice format. The data is provided by Chicago Department of Planning and Development

If you want to learn more about the dataset please visit the official Dataset website

How to use the app:

Navigate using the bar at the top. In order to search for specific data use the form. Otherwise you can view unfiltered data in the data tab.

Click on the map tab if you want to view the data in a nicer mapped format to get a better sense of approximate locations of land

About Page

This app allows users to view land inventory owned by the city of Chicago. User can view either in list format or map format
